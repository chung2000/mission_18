# 🎬 [Mission 18] AI 기반 영화 리뷰 감성 분석 서비스 구축
웹 애플리케이션 : 프론트엔드는 Streamlit, 백엔드는 FastAPI

---

## 🚀 1. 소개
**영화 정보 관리, 사용자 리뷰 등록, 그리고 리뷰 감성 분석**을 통합한 웹 애플리케이션을 개발합니다.

* **Frontend**: Streamlit
* **Backend**: FastAPI
* **AI Model**: Sentiment Analysis (감성 분석)

---

## 🎨 2. 프론트엔드 (Streamlit)

### ✅ 기본 기능
* **영화 목록 표시**: 영화 제목, 포스터 이미지, (옵션) 평균 평점 시각화
* **영화 추가**: 제목, 개봉일, 감독, 장르, 포스터 URL 입력 폼 제공

### 🔥 심화 기능 (선택 사항)
* **리뷰 등록**: 데이터베이스에 저장된 영화를 선택하여 리뷰 작성
* **실시간 감성 분석**: 리뷰 작성 후 AI 모델을 통한 분석 결과 즉시 표시
* **리뷰 히스토리**: 최근 10개 리뷰 표시 (영화 ID, 등록일, 내용, 분석 결과)

### 🌐 배포 및 주의사항
* **배포**: Streamlit Cloud 활용
* **데이터 관리**: 모든 데이터는 백엔드(FastAPI)에서 관리하며, 프론트엔드 내부에는 별도로 저장하지 않음

---

## ⚙️ 3. 백엔드 (FastAPI)

### ✅ 기본 기능
* **영화 관리 API**:
    * 등록: 제목, 개봉일, 감독, 장르, 포스터 URL (나무위키 등 참고)
    * 조회: 전체 목록 및 특정 영화 상세 조회
    * 삭제: 특정 영화 정보 삭제

### 🔥 심화 기능 (선택 사항)
* **리뷰 관리**: 리뷰 등록, 조회, 삭제 기능
* **통계 조회**: 감성 분석 점수를 기반으로 한 평균 평점 계산
* **모델 서빙**:
    * 적절한 감성 분석 모델 리서치 및 적용
    * **모델 경량화** 방식에 대한 고민 및 최적화
